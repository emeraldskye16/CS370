# CS370
This project demonstrates the application of reinforcement learning and neural networks to a real-world problem by training an intelligent agent to navigate a maze. The agent, represented as a pirate, operates on an 8×8 grid and learns to efficiently locate a treasure while avoiding obstacles and inefficient routes. Using Deep Q-Learning (DQN), the agent improves its performance over time by learning from interactions with the environment and maximizing rewards.

In this project, I was provided with a Jupyter Notebook that included the environment's foundational structure, such as the maze configuration and the general framework for training an agent. My responsibility was to implement the core learning components, including the Deep Q-Learning algorithm, neural network model, and training logic. I also applied an epsilon-greedy strategy to balance exploration and exploitation, allowing the agent to initially explore its environment randomly before gradually relying on learned behaviors. Additionally, I incorporated experience replay to improve training stability and performance. Through these contributions, I transformed the agent from making random moves into consistently finding the optimal path to the treasure.
